# Wi-Fi Optimizer

This project is a web application designed to help non-technical users analyze and optimize their local Wi-Fi networks. It addresses common issues like channel interference and poor signal strength by providing clear diagnostics and recommendations.

The key feature is an integrated **LLM (AI) assistant** that translates complex network data into simple, step-by-step instructions for improving network performance (e.g., changing router channels or physical placement).

*This application is being developed as part of a Master of Science in Computer Science.*

Follow these instructions to get a copy of the project up and running on your local machine for development and testing.

### Prerequisites

You will need [Node.js](https://nodejs.org/) (v18 or higher) installed on your machine.

### Installation & Running

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/DariiaBabii/wifi-optimizer-app.git](https://github.com/DariiaBabii/wifi-optimizer-app.git)
    cd wifi-optimizer-app
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:5173](http://localhost:5173) (or the URL shown in your terminal) in your browser to see the application.
